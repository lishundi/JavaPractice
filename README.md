* Fibonacci.java<br>
算法思想：<br>
第一步：sum = 第一个数f1+第二个数f2;<br>
第二步：获取第三个数ｆ = f1+f2;<br>
第三步：sum = 第一个数f1+第二个数f2+第三个数f;<br>
第四步：将第二个数f2赋值给第一个数f1<br>
第五步：将第三个数f赋值给第二个数f2<br>
然后循环上面步骤
